# Tire Monitor

실시간 타이어 압력 및 마모 모니터링 앱

## 기능

- 실시간 타이어 압력 모니터링
- 타이어 마모 상태 추적
- 이상 상태 알림
- 사용자 정의 임계값 설정
- 직관적인 UI로 쉽게 상태 확인

## 개발 환경 설정

1. Flutter SDK 설치
   - Flutter SDK를 [공식 웹사이트](https://flutter.dev/docs/get-started/install)에서 다운로드하여 설치합니다.

2. 프로젝트 클론
   ```bash
   git clone [repository-url]
   cd tire_monitor
   ```

3. 의존성 설치
   ```bash
   flutter pub get
   ```

4. 앱 실행
   ```bash
   flutter run
   ```
